Cadastro de Notificações de Medalhas

Para se cadastrar no sistema de notificações de medalhas das Olimpíadas, siga as instruções abaixo:

  1. Clique no link: Notificação Olímpica Bot ou escaneie o QR Code abaixo.
  2. Você será redirecionado para uma conversa com o bot de notificações.
  3. Basta enviar uma mensagem ao bot e ele responderá com orientações para o cadastro no sistema.
